Pengenalan HTML

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat dokumen hypertext. HTML tidak membedakan huruf besar dengan kecil (kecuali versi XHTML strict).

Struktur Dasar HTML


<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
    <title>Judul Halaman</title>
</head>
<body>
    <!-- Konten halaman di sini -->
</body>
</html>
